# Command Line Notes

## Introduction to the Command Line
The command line is a text-based way to navigate the computer and work with files and folders.

## First commands
- `pwd` shows where I am
- `ls` shows what is here
- `cd` moves between directories

## Interacting with the Command Line

Common commands practiced:

- `ls` list files
- `ls -a` show hidden files
- `cd` change directory
- `mkdir` create directories
- `touch` create files
